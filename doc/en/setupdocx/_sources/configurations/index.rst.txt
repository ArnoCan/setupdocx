List Available Templates
========================
The command *build_docx* provides the option *--list-templates*.
This displays a list of all provided templates, see also options
:ref:`--list-templates-std <setupdocxbuild_OPTIONS_list_templates_std>`
and
:ref:`--list-templates= <setupdocxbuild_OPTIONS_list_templates>`.

For standar list with names only:

   .. parsed-literal::
   
      python setup.py build_docx :ref:`--list-templates-std <setupdocxbuild_OPTIONS_list_templates_std>`

For extended list with parameters   

   .. parsed-literal::
   
      python setup.py build_docx :ref:`--list-templates="display=abs;" <setupdocxbuild_OPTIONS_list_templates>`
