5.4. List Available Templates¶

The command build_docx provides the option –list-templates. This displays a list of all provided templates, see also options –list-templates-std and –list-templates=.

For standar list with names only:

python setup.py build_docx –list-templates-std

For extended list with parameters

python setup.py build_docx –list-templates=”display=abs;”
